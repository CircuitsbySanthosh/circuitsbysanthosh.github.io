Posture Correction System

Real-time posture monitoring system using dual MPU6050 sensors, ultrasonic-based activation and intelligent feedback indicators.

Arduino Uno Embedded Systems 2025

Problem

Poor sitting posture is common among students and professionals, leading to long-term neck and back issues. Continuous posture awareness can prevent chronic spinal problems.

Objective

To design a low-cost, power-efficient embedded system that detects incorrect posture in real time and alerts the user instantly.

System Overview

Two MPU6050 sensors are placed on upper and lower body. The system activates only when ultrasonic sensor detects a user continuously for 5 seconds.
