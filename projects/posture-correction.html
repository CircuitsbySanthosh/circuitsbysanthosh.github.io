Posture Correction System

A real-time posture monitoring system built using dual MPU6050 sensors, ultrasonic-based activation, and visual feedback indicators to promote healthier sitting habits.

Problem

Poor sitting posture is common among students and professionals, leading to long-term neck and back issues. Continuous posture awareness can prevent chronic spinal problems.

Objective

To design a low-cost, power-efficient embedded system that detects incorrect posture in real time and alerts the user instantly.

Solution Overview

The system uses two MPU6050 sensors to monitor upper and lower body angles. An ultrasonic sensor activates the system only when a person is detected for more than 5 seconds to save power.
